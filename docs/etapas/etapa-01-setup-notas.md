## Tareas

- [ ] Leer la página [Sistema de notas](../plan/notas.md) (~15 min)
- [ ] Elegir **un** workflow: papel · Notion · híbrido (recomendado)
- [ ] Comprar materiales si elegiste papel/híbrido (cuaderno A5 + fichas + carpeta · 30-60 €)
- [ ] Configurar setup digital si elegiste Notion/Obsidian (gratis)
- [ ] Poner fecha de inicio en el [panel](../#mi-panel)

## Cómo se estudia esta etapa

!!! tip "Elegí YA, optimizá después"
    No pierdas una semana comparando setups. Híbrido (papel + Notion) funciona para el 80% de la gente. Si dudás, empezá por ahí — podés migrar más adelante sin perder nada.

Sistema listo para empezar a leer. **NO LEER LIBROS TODAVÍA.**
