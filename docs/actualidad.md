# Actualidad

Últimos titulares de fuentes que dictan agenda en ciencia política y geopolítica. Actualizado automáticamente cada día (vía GitHub Actions).

!!! note "Primera ejecución"
    Esta página la actualiza un workflow programado. La primera vez se rellenará en la próxima ejecución (~24 h después del primer push). Para ejecutarla manualmente: ve a [Actions → Update news feeds](https://github.com/MrSpagVi/Libros-Politica/actions/workflows/news.yml) y aprieta "Run workflow".

## Fuentes monitorizadas

- **Real Instituto Elcano** (España) — política exterior y geopolítica.
- **Nueva Sociedad** (Argentina) — análisis político-social latinoamericano.
- **The Conversation — Politics & Society** — análisis académico global.
- **CIDOB** — Barcelona Centre for International Affairs.
- **El Salto Diario** — periodismo crítico español.

Si una fuente está caída ese día, la página la salta sin romperse.
