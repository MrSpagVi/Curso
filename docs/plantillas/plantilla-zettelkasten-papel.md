# Plantilla — Fichas Zettelkasten papel

> El sistema original de **Niklas Luhmann** (1952-1997): 90.000 fichas, 70 libros publicados, 600 artículos. **Tu cuaderno guarda lo que leíste; tus fichas guardan lo que piensas.**

---

## :material-package-variant: Material que necesitas

- **Fichas A6 blancas** (105 × 148 mm). ~10€ por pack de 500 en cualquier papelería. Bastan para 12-18 meses.
- **Caja o fichero:**
  - Económico: caja de zapatos forrada (gratis si tienes la caja).
  - Decente: fichero rígido de cartón con separadores (~15€).
  - Bonito: fichero de madera tipo "wooden index card box" (~30-50€ en Amazon).
- *Opcional:* separadores con pestaña para dividir por números (1-99, 100-199, etc.).

**Total: 10-50€ una sola vez.**

## :material-numeric: El sistema de numeración Luhmann

Este es el truco que hace el sistema durable. **Te permite intercalar fichas en cualquier momento sin romper la estructura existente.**

### Reglas

1. **Tu primera ficha es `1`.**
2. **Tu segunda ficha INDEPENDIENTE es `2`.** (Es decir, sobre un tema distinto.)
3. **Si una ficha COMENTA o PROFUNDIZA otra:**
   - La que profundiza a `1` es `1a`.
   - La siguiente que profundiza a `1` (después de `1a`) es `1b`.
   - Si quieres meter algo ENTRE `1a` y `1b`, es `1a1`.
   - Una contra-réplica a `1a1` es `1a1a`.
4. **No hay límite de profundidad.** Pero rara vez vas más allá de `1a2b3`.

### Ejemplo de crecimiento orgánico (~15 fichas iniciales)

```
1   - "El estado de naturaleza hobbesiano es ficción metodológica"
1a  - "Diferencia con Locke: Hobbes pesimista, Locke utilitarista"
1a1 - "Pero ambos comparten el supuesto del individuo aislado"
1a2 - "Rousseau rompe con esto: el contrato CREA al sujeto"
1b  - "Aplicación a un caso actual: Iraq 2003 como estado de naturaleza"
2   - "Concepto de soberanía en Schmitt: quien decide la excepción"
2a  - "Schmitt vs Weber: legitimidad legal-racional vs decisionismo"
2b  - "Agamben extiende Schmitt: estado de excepción permanente"
2b1 - "Mbembe: la necropolítica como caso extremo"
3   - "Marx, teoría del valor-trabajo"
3a  - "Cap. 1 de El Capital: forma simple del valor"
3a1 - "Crítica: ¿la teoría del valor es defendible hoy?"
3b  - "Acumulación originaria: cap. XXIV"
3b1 - "Aplicación al despojo de tierras indígenas en LATAM"
3b2 - "Mariátegui ve esto como permanente, no histórico"
```

### Lo brillante del sistema

Si dentro de 2 años tienes una idea nueva sobre la conexión Hobbes-Locke (entre `1a` y `1a1`), la insertas como `1a0`. **No tienes que renumerar nada.** El orden lógico se mantiene.

## :material-card-bulleted: Reglas inviolables de las fichas

1. **Una idea por ficha.** Si son dos, parte la ficha en dos.
2. **Título declarativo:** una **afirmación completa**, no un sustantivo. ❌ "Justicia" ✅ "La justicia distributiva presupone un sujeto monolítico que el feminismo refuta".
3. **En tus palabras.** Si solo copias del autor, no hay ficha — eso es nota de lectura, no permanent.
4. **Mínimo 2 conexiones a otras fichas** explícitas (`→ Xa`, `→ Yb`). Sin enlaces, la ficha está muerta en el archivo.
5. **Fuente y fecha al pie.** Para poder reconstruir contexto en 5 años.
6. **Si se te acaba espacio en la ficha, NO la sigas en el reverso.** Empieza otra ficha (`1a2`) que continúe la anterior.

## :material-card-text: Tipos de fichas especiales

### Ficha de **estructura/índice** (mapa temático)

Una ficha especial que NO es una idea, sino un MAPA de fichas conectadas.

```
┌─────────────────────────────────────────┐
│ I-JUST                  [Index card]     │
│                                          │
│ JUSTICIA — fichas relacionadas          │
│                                          │
│ Posición original / velo de ignorancia: │
│   12, 12a, 12a1, 12a2                   │
│                                          │
│ Crítica comunitarista (Sandel):         │
│   12a1, 14, 14a                         │
│                                          │
│ Crítica interseccional (Crenshaw):      │
│   12b, 23                               │
│                                          │
│ Crítica decolonial (Quijano):           │
│   7c, 7c1                               │
│                                          │
│ Justicia libertarian (Nozick):          │
│   18, 18a                               │
│                                          │
│ Mi posición provisional:                │
│   → ver ficha 56 "Síntesis sobre justicia"│
└─────────────────────────────────────────┘
```

Las fichas de índice empiezan con `I-` o las pones en una sección separada del fichero.

### Ficha de **autor**

Sintetiza la posición de un autor en una sola ficha:

```
┌─────────────────────────────────────────┐
│ A-RAWLS                [Author card]     │
│                                          │
│ RAWLS, John (1921-2002, Harvard)        │
│                                          │
│ Tesis central: la justicia es la primera│
│ virtud de las instituciones sociales.   │
│ Se elige imparcialmente desde la        │
│ posición original detrás del velo de    │
│ ignorancia.                              │
│                                          │
│ Concepto clave: principios de justicia: │
│  1) iguales libertades básicas          │
│  2) desigualdades aceptables solo si    │
│     benefician a los más desfavorecidos │
│                                          │
│ Libros leídos: Teoría de la Justicia    │
│ (1971, sel. caps I y III, mes 14)       │
│                                          │
│ Mis fichas sobre Rawls:                 │
│   12, 12a, 12a1, 12a2, 12b              │
│                                          │
│ Dialoga con: Sandel, Nozick, Habermas,  │
│   Crenshaw, Quijano                     │
└─────────────────────────────────────────┘
```

## :material-magnify: Cómo encontrar una ficha cuando la necesitas

El sistema parece desorganizado pero funciona porque:

1. **Las fichas de índice (I-XXX)** te llevan a clusters temáticos.
2. **Las fichas de autor (A-XXX)** te llevan a clusters por pensador.
3. **Las conexiones explícitas (`→ 12a`)** te llevan de unas a otras.

**No necesitas catalogación exhaustiva como una biblioteca.** Necesitas suficientes puntos de entrada para encontrar **cualquier** ficha en 2-3 saltos.

Luhmann tenía ~90.000 fichas y las encontraba todas. Tu objetivo: ~500-1500 fichas en 28 meses, totalmente manejable.

## :material-clipboard-text: Rutina semanal

**Sábado o domingo (60-90 min):**

1. Abre tu cuaderno de la semana. Lee las literature notes.
2. Identifica 3-7 ideas que TE inspiraron a pensar (no que el autor dijo — que TÚ pensaste al leerlo).
3. Para cada una:
   - Escribe la ficha siguiendo la plantilla.
   - Numera siguiendo el sistema Luhmann (¿es nueva línea? ¿profundiza una existente?).
   - Archiva en la caja.
   - Actualiza la ficha de índice (o crea una nueva si es tema nuevo).
4. Si el tema acumula 5+ fichas, crea o actualiza la **ficha de índice** correspondiente.

**Tiempo realista por ficha:** 5-10 minutos. Es **escritura lenta y deliberada**. NO copies — piensa mientras escribes.

## :material-help-circle: FAQ

??? question "¿Qué pasa si quiero mover una ficha de lugar?"
    No la muevas físicamente. Cambia las **conexiones explícitas** en otras fichas para reflejar el nuevo papel de esa ficha. La numeración original se queda. (Sí, suena raro al principio. Luhmann lo defendió 47 años.)

??? question "¿Y si me equivoco en una ficha?"
    **No la tires.** Marca el error y escribe una ficha nueva (`1a3`, por ejemplo) que diga "corrección a 1a: estaba equivocado porque...". El error es parte del archivo.

??? question "¿Cuántas fichas por libro?"
    Idealmente 5-15 para un libro mayor. Si haces 50, estás resumiendo, no pensando. Si haces 1, no profundizaste.

??? question "¿Y si pierdo la caja en un incendio?"
    **Fotografía las fichas mensualmente** y sube las fotos a Google Drive o iCloud. Tiempo: 10 min/mes. La pérdida total es prevenible.

??? question "¿Puedo digitalizar las fichas al final?"
    Sí. Cuando termines el plan (mes 28), puedes transcribir las fichas a Obsidian o Notion para tenerlas digitales. Pero **no las digitales mientras lees** — perderías la ventaja cognitiva del papel.
